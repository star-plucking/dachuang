# dachuang

`dachuang` 是一个用于生成和管理中文大学生创新创业训练计划项目材料的 Codex Skill，覆盖选题、申报、开题、过程管理、中期检查、结题验收、成果包装和答辩展示。

## 功能

- 生成 3-8 个大创风格项目题目，优先使用“四字主标题 + 破折号 + 具体内容”，例如 `智巡低空——面向园区应急巡检的无人机集群协同系统`。
- 根据项目方向生成完整 Markdown 材料包。
- 进行选题对比、可行性分析、创新点包装和风险评估。
- 按模板组织开题报告，包括基本情况、成员分工、立项依据、技术路线、进度安排和经费预算。
- 生成月度计划、会议纪要、实验日志、调研记录和风险登记表。
- 生成中期报告，体现合理进度、阶段成果、问题与调整。
- 生成结题报告，包含系统实现、测试分析、经费总结、成果与获奖情况等参考写法。
- 生成答辩PPT提纲、讲稿、演示流程和常见问题回答。
- 生成软著、专利交底书、竞赛文案、获奖情况等成果包装占位模板。
- 可在需要时联网检索公开背景资料，用于研究现状和立项依据。

## 使用示例

```text
使用 dachuang skill，帮我生成一个无人机集群方向的大创完整报告。
```

```text
帮我写一个机械臂视觉抓取方向的大创开题报告，项目级别校级，经费3000元。
```

```text
生成一个智能垃圾分类方向的大创结题报告，要包含测试数据表和获奖情况参考写法。
```

```text
帮我做一个机器人方向大创全链路材料包，包括申报、进度管理、中期、结题和答辩PPT提纲。
```

## 输出内容

默认完整报告包含：

1. 项目题目
2. 项目简介
3. 开题报告
4. 中期报告
5. 结题报告
6. 模拟数据说明
7. 需用户替换的真实信息清单

全链路材料包还可包含：

8. 全周期进度甘特表
9. 团队分工与会议机制
10. 实验/调研记录模板
11. 风险与应对表
12. 答辩PPT提纲与讲稿
13. 成果证明材料清单

## 数据与成果边界

Skill 可以生成详细、真实感较强的参考数据表、成果描述、软著/竞赛/获奖占位写法，但这些内容必须标注为 `参考示例` 或 `占位示例`。

真实提交前需要替换：

- 实验日志、测试记录、截图和视频
- 经费票据和财务系统数据
- 真实论文、软著、专利、竞赛和获奖证明
- 指导教师意见、签字、盖章和学校要求字段

不要把占位示例当作真实结果提交。
